## Key Risk Findings & Mitigations

### 1. AI Prompt Injection
- **Finding:** LLM narrative vulnerable to instruction override.
- **Mitigation:** Switch to "System Prompt" architecture + output validation.

### 2. Tenant Isolation
- **Finding:** Potential for lateral movement if K8s namespace breached.
- **Mitigation:** Enforcing GKE Workload Identity and Network Policies (Hardening Guide).

### 3. Disaster Recovery
- **Plan:** Confirmed 15-minute RTO target using cross-region Cloud SQL replication (`us-east1` -> `us-west1`).
